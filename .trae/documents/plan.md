# TOP投诉小区隐患整治系统 - 轻量级实施计划

## 项目概述
基于已批准的PRD，使用最轻量级技术栈快速实现TOP投诉小区隐患整治系统。单HTML文件 + 原生JS + Tailwind CDN，无需构建工具。

## 技术栈
- 单HTML文件应用（无构建工具）
- Tailwind CSS CDN
- 原生 JavaScript（无框架）
- LocalStorage 数据持久化

## 实施步骤

### 阶段一：项目结构搭建
1. 创建 index.html（主入口，包含所有HTML结构）
2. 引入 Tailwind CSS CDN
3. 创建 app.js（所有JS逻辑）

### 阶段二：数据层
4. 定义数据模型（隐患工单、投诉明细、用户）
5. 创建Mock数据生成函数
6. 实现 LocalStorage 读写封装

### 阶段三：核心功能实现
7. 登录模块：角色选择（地市/区县管理员）
8. 仪表盘：统计卡片 + 待办列表
9. 隐患工单列表：表格 + 筛选 + 分页
10. 隐患工单详情：信息展示 + 流程操作 + 时间线
11. 投诉工单明细：表格 + 筛选 + 导出

### 阶段四：状态流转
12. 实现工单状态机（待接单→方案制定→实施中→审核中→已归档）
13. 实现挂起/解挂功能
14. 实现历史进展自动记录

### 阶段五：导出功能
15. 实现隐患工单Excel导出
16. 实现投诉工单明细Excel导出

### 阶段六：测试验证
17. 浏览器打开验证所有功能
18. 验证两地市角色权限差异
19. 验证完整工单流程
